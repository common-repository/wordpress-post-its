== Description ==

Widget that let you put messages in a post-it style. You can add many messages as you want,
and the messages will appear one after another in a period of time. The time interval
is defined by you, and, you can set a expires date to disable the message or messages automatically
 

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the 'wordpress_postits' folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.
4. After yo activate the plug, you will see an new option called "WPost-Its !" in the "Settings" menu.

== Adding message ==
5. Go to the "Post-Its !" option in the Settings menu.
6. Click in the "Add New Post-It"

== Configuring the Widget ==
   
7. Once you have added messages, go to the "Widgets" section in the "Design" menu. 
8. Look for the "WordPress Post-its" widget an clik in the "add" link.
9. When the widget was added to the click on the "edit" link.
10. There, you can configure the time thats one message will stay in the screen
    before another message appears, and the style of the post-it.
11. Press "Save Changes". Done !
